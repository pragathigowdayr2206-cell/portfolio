## 🚀 Deployment Options

This portfolio supports multiple deployment platforms:

### 1. GitHub Pages (Recommended)
- **Automatic**: GitHub Actions deploys on every push to main
- **Manual**: Use `npm run deploy` or the deployment scripts
- **URL**: `https://pragathigowdayr2206-cell.github.io/portfolio`

### 2. Netlify
- Connect your GitHub repository to Netlify
- Configuration file: `netlify.toml`
- Features: Form handling, analytics, custom domains

### 3. Vercel
- Connect your GitHub repository to Vercel
- Configuration file: `vercel.json`
- Features: Global CDN, preview deployments

### 4. Other Platforms
The site is static HTML/CSS/JS, so it works on any web hosting service.

### Local Development

1. Clone the repository:
```bash
git clone https://github.com/pragathigowdayr2206-cell/portfolio.git
cd portfolio
```

2. Install dependencies:
```bash
npm install
```

3. Start development server:
```bash
npm start
```

4. Open [http://localhost:3000](http://localhost:3000) in your browser.

### Deployment Pipeline

The project uses GitHub Actions for continuous deployment:

- **Trigger**: Push to `main` branch
- **Build**: Static site (no build required)
- **Deploy**: Automatic deployment to GitHub Pages

### Deployment Scripts

For easy deployment, use the provided scripts:

**Windows:**
```bash
deploy.bat
```

**Linux/Mac:**
```bash
./deploy.sh
```

These scripts provide options to:
- Deploy directly to GitHub Pages
- Push changes to GitHub (triggers automatic deployment)
- Check deployment status
- Install dependencies
